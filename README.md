# Alquimia
DESCRIPCIÓN:
- Este proyecto consiste en el desarrollo de una landing page, o web page, para un emprendimiento familiar de licores artesanales. El objetivo principal fue crear un sitio web simple, atractivo y adaptable a distintos dispositivos, que permita presentar los productos y transmitir la identidad artesanal del proyecto.

DESARROLLO:
- El trabajo comenzó con la estructura base en HTML, organizando el contenido en secciones claras: encabezado, introducción, productos, recursos y pie de página. Se utilizaron etiquetas semánticas para mejorar la organización y la legibilidad del código. Además, se integraron herramientas externas como Bootstrap para facilitar el diseño responsive, Google Fonts para mejorar la tipografía y Font Awesome para incorporar íconos.

- En cuanto a los estilos, se trabajó con un archivo CSS externo, aplicando un reset básico y definiendo estilos generales para textos y contenedores. Uno de los puntos más importantes del desarrollo fue el uso de Flexbox, que permitió resolver la alineación de elementos y organizar correctamente las tarjetas de productos, logrando una distribución más prolija y flexible.

- La sección de productos se diseñó mediante tarjetas reutilizables que incluyen imagen, título, descripción y botón. Se aplicaron mejoras para mantener consistencia visual y asegurar que todos los elementos se adapten correctamente al contenido.

- El diseño responsive se logró mediante media queries, ajustando tamaños, distribución y visibilidad de elementos en pantallas más pequeñas. Esto permitió mejorar la experiencia de usuario en dispositivos móviles.

APRENDISAJE Y PROCESO:
- El desarrollo fue iterativo, con múltiples pruebas y ajustes. Se investigaron distintas soluciones, se probaron enfoques alternativos y se corrigieron errores a lo largo del proceso. Se utilizaron recursos como documentación online y ejemplos prácticos para reforzar el aprendizaje.

TECNO LOGÍAS UTILIZADAS:
- HTML5
- CSS3
- Bootstrap 5
- Google Fonts
- Font Awesome

CONCLUSIÓN:
- El proyecto permitió afianzar conocimientos en maquetado web, diseño responsive y uso de herramientas modernas de desarrollo. Además, ayudó a comprender la importancia de la práctica, la iteración y la mejora continua en la construcción de una web funcional y estética.
